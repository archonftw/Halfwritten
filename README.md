# HalfWritten ✍️

**HalfWritten** is a modern full-stack writing and storytelling platform where users can create, publish, explore, and interact with written content.  
It is designed as a clean, minimal, and engaging space for writers and readers to share ideas, unfinished thoughts, stories, and creative expression.

## 🚀 Features

- 🔐 **Authentication with Clerk**
  - Secure sign-up and sign-in flow
  - Protected routes for authenticated users
  - Personalized user experience

- 📝 **Create & Publish Posts**
  - Users can write and publish their own content
  - Clean and distraction-free writing experience

- 📚 **Explore Community Posts**
  - Browse content shared by other users
  - View detailed post pages

- 💬 **Comment System**
  - Users can engage with posts through comments
  - Interactive community experience

- ❤️ **Like / Engagement Features**
  - Users can react to posts
  - Encourages content discovery and interaction

- 🗑️ **Post Management**
  - Authors can manage or delete their own posts
  - Ownership-based actions

- 🎨 **Modern UI/UX**
  - Responsive and visually appealing design
  - Smooth layout built for readability and focus
